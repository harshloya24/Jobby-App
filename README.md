# 💼 Jobby App (Full Stack Job Portal)

A full stack job portal application where users can search, filter, and explore jobs with secure authentication and a responsive UI.

---

## 🚀 Live Features

* 🔐 User Login Authentication (JWT-based)
* 🔍 Search Jobs functionality
* 🎯 Filter jobs by Employment Type (Full Time, Part Time, Internship)
* 💰 Filter by Salary Range
* 📄 Detailed Job View Page
* 🚫 Protected Routes (unauthorized access restricted)
* ⚡ Fast and responsive UI

---

## 🛠️ Tech Stack

### Frontend

* React.js
* React Router (v6)
* CSS

### Backend

* Node.js
* Express.js

### Database

* SQLite

### Authentication

* JWT (JSON Web Token)

---

## 📂 Folder Structure

```
client/   → React frontend

```

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```
git clone https://github.com/harshloya24/Jobby-App.git
```

---

### 2️⃣ Setup Frontend

```
cd client
npm install
npm start
```

Frontend will run on:

```
http://localhost:3000
```

---

### 3️⃣ Setup Backend

```
cd server
npm install
node server.js
```

Backend will run on:

```
http://localhost:5000
```
